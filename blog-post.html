How to Add a New Post

To publish a new post:

  1. Copy this file (blog-post.html) and rename it.
  2. Update the <title>, meta description, and Open Graph tags in the <head>.
  3. Replace the title, date, and tags in the <header>.
  4. Write your content inside the .post-prose div.
  5. Add a card linking to it in blog.html.

The prose styles handle headings, paragraphs, lists, code blocks, blockquotes, images, and horizontal rules out of the box.
