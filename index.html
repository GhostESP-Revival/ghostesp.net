Capabilities

The largest update in the project's history turns an ESP32 into a full wireless research platform. Pick a band to see what it can do. A NEW tag marks what's new in v2.0.

WiFi Suite

2.4 & 5 GHz testing, capture, and monitoring.

  • Deauth, beacon spam & karma
  • Evil portal & captive-portal creds
  • WPA3 / SAE testing + compliance checker
  • PCAP logging & live Wireshark over USB
  • WiFi airspace monitor NEW
  • Recon: ARP · SSH · SNMP · NetBIOS · HTTP NEW
  • GPS wardriving & WiGLE export
  • mDNS discovery & OUI vendor lookup

Bluetooth LE

Scan, capture, and spoof BLE devices.

  • BLE scan & raw scanner
  • Device spam modes
  • AirTag scan / spoof · Flipper Zero finder
  • GATT enumeration · BLE → Wireshark
  • Skimmer & surveillance (Flock) detection
  • BLE wardriving & WiGLE export

NFC & Infrared

Read, write, and replay NFC and IR signals.

  • PN532 · Chameleon Ultra (BLE control)
  • NTAG read/write
  • MIFARE dictionary + default-key attack
  • Flipper .nfc import/export & parser set
  • IR learn / capture / replay
  • Universal IR library · Flipper .ir support

SubGHz & RF

Analyze, capture, and replay across the bands.

  • CC1101 scan / replay · 315–915 MHz
  • Waterfall spectrum analyzer
  • SubGHz protocol decoders · Flipper .sub
  • NRF24 spectrum analyzer & sniff
  • Zigbee / 802.15.4 capture
  • Passive jamming detection

Network & Ethernet

Wired attacks and live traffic interception.

  • Ethernet / W5500
  • ARP poisoning & MITM tools
  • DNS intercept / sinkhole (NXDOMAIN)
  • Credential capture: TLS SNI · HTTP · FTP
  • DIAL / Chromecast · network printer (PJL)
  • Camera streaming & motion webhooks

BadUSB & HID

HID payloads and host-side control.

  • BadUSB / DuckyScript
  • VID/PID identity options
  • USB keyboard host mode
  • Trackpad cursor & HID output NEW
  • Mouse jiggler NEW
  • Redesigned WebUI NEW

Platform & UI

Control, extend, and theme the whole device.

  • LVGL UI · 60 FPS · 17+ themes NEW
  • Touch / keyboard / encoder · web dashboard
  • File manager · screen mirroring
  • Native SD apps: .gapp, App Gallery, SDK, XIP on C5 NEW
  • Asset packs: .gtheme live swap NEW
  • Ghostchi: 50 levels & moods NEW
  • PIN lockscreen & setup wizard NEW
  • GhostLink dual-ESP · Android app · Flipper companion
